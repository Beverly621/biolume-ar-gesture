# Security Policy

## Supported Versions

We currently support security updates for the latest version of this project.

| Version | Supported |
| ------- | --------- |
| latest  | ✅ |
| older versions | ❌ |

## Reporting a Vulnerability

Please do not report security vulnerabilities through public GitHub issues.

If you discover a vulnerability, please report it privately by emailing:

Liny629@outlook.com & kimbeverly629@gmail.com

Please include:

- A description of the vulnerability
- Steps to reproduce the issue
- Potential impact
- Any suggested fix, if available

We will try to acknowledge your report within 7 days and provide updates as we investigate.

## Disclosure

Please give us reasonable time to investigate and fix the vulnerability before making it public.
