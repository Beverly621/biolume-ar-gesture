## 2. 核心视觉定位

项目整体不走硬科技 HUD / 电网风，而是“夜间森林中的柔光生命体”：蓝绿发光、半透明花瓣、微湿润水滴、孢子漂浮、藤蔓缓慢呼吸。用户的手势不是在触发按钮，而是在唤醒一个会生长、会回应触碰、会呼吸的微型生态系统。

## 3. 配色方案

| 用途 | 色彩名 | Hex | 使用位置 |
|---|---|---:|---|
| 主发光蓝绿 | Cyan Glow | `#7EF6E7` | 花瓣边缘、藤蔓脉络、孢子主光 |
| 深夜蓝 | Night Indigo | `#102A3A` | 暗部背景、阴影、遮罩过渡 |
| 柔和苔藓绿 | Moss Soft Green | `#8FD6A3` | 苔藓、叶芽、生态斑块 |
| 花蕊暖光 | Pollen Warm Yellow | `#F8D879` | 花蕊、触点高光、粒子中心 |
| 薰衣草紫 | Lavender Aura | `#A78BFA` | 花瓣内侧、双手能量网次级光 |
| 低饱和雾白 | Mist White | `#DDF7EF` | 水滴高光、透明花瓣内反光 |

色彩逻辑：以低饱和夜蓝为底，蓝绿作为生命荧光主色，暖黄只出现在花蕊、种子、触点中心，紫色作为少量奇幻过渡，避免过度霓虹化。

## 4. 手势交互逻辑

### 动作 A：张开手掌｜掌心花开

| 模块 | 逻辑 |
|---|---|
| 触发条件 | 检测手掌从半握拳到张开，五指展开角度增大；建议 opening ratio > 0.65 且持续 0.2 秒后触发。 |
| 一触即发 | 掌心中央出现发光种子，0.15 秒内从暖黄色光点凝聚为花苞。 |
| 延迟变化 | 0.25–0.8 秒内花瓣分三层展开：内层先开，外层延迟 0.15 秒，最后花蕊释放孢子。 |
| 粒子表现 | 花粉从花蕊向外扩散，颜色由 `#F8D879` 过渡到 `#7EF6E7`，生命周期 1.8–3 秒。 |
| 空间布局 | 花朵中心在掌心上方 3–5 cm；花朵直径约为掌宽 45%–60%；孢子半径为掌宽 1.2–1.8 倍。 |

### 动作 B：指尖轻触｜指尖种苔藓 + 水滴波纹

| 模块 | 逻辑 |
|---|---|
| 触发条件 | 食指指尖与目标平面距离 < 2–3 cm，或食指与拇指轻捏后释放。 |
| 一触即发 | 指尖落点出现一滴暖黄色水珠，水珠压扁成圆形波纹。 |
| 延迟变化 | 0.2 秒后波纹边缘长出苔藓粒；0.5 秒后生成 1–3 片迷你荷叶；0.8 秒后出现露珠高光。 |
| 粒子表现 | 触点周围 12–30 个微粒环形扩散，部分落点变成小叶芽。 |
| 空间布局 | 单个生态斑块直径约为指尖宽度 6–10 倍；水波纹最大半径约 8–12 cm。 |

### 动作 C：双手合十再拉开｜藤蔓拉扯 + 孢子能量网

| 模块 | 逻辑 |
|---|---|
| 触发条件 | 双手掌心距离 < 8–12 cm 且停留 0.3 秒判定为合十蓄能；随后双手距离扩大到 25–60 cm 进入拉开状态。 |
| 一触即发 | 合十时两掌之间出现低亮度孢子云，像被手掌保护的种子团。 |
| 延迟变化 | 拉开时孢子云被拉成长条，生成 5–9 条主藤蔓线，并从主线上长出细菌丝分支。 |
| 粒子表现 | 发光孢子沿线缓慢流动；双手距离越大，流速越快，线条张力越明显。 |
| 空间布局 | 能量网中心位于两掌连线中点；宽度等于双手距离；高度约为掌宽 1.2–1.5 倍。 |

## 5. 技术实现提示

| 效果 | 推荐技术 |
|---|---|
| 花苞展开 | GLTF 骨骼动画 / Morph Target / 顶点动画 Shader |
| 藤蔓沿手指生长 | Curve Path + Tube Geometry Reveal / Geometry Nodes 预烘焙 / 骨骼绑定 |
| 叶芽冒出 | InstancedMesh 小叶片，scale 0→1，随机旋转与延迟 |
| 发光脉络流动 | Emissive Texture + UV Scrolling + Fresnel 边缘光 |
| 孢子漂浮 | Particle System / THREE.Points / GPU Instancing |
| 水滴波纹 | Radial Ripple Fragment Shader + Normal Map Distortion |
| 苔藓扩散 | Procedural Noise Mask + Alpha Reveal + Instanced Grass Cards |
| 双手能量网 | 动态曲线线段 + 有机 Noise 偏移 + 粒子吸附在线段上 |

## 6. 平台建议

- **Effect House**：适合 TikTok 风格滤镜、手势触发、轻量粒子、快速预览。
- **Lens Studio / Spark AR 迁移方向**：适合社交平台滤镜，但需确认当前发布生态与平台政策。
- **WebXR + Three.js**：适合高级作品集 Demo、Quest / 浏览器展示、复杂手部关节驱动与 Shader 效果。
